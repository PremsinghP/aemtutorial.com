Goto Tools > Operations > Diagnosis > Log Messages. Configure DEBUG log level on com.day.cq.search to monitor search queries. B. Goto Tools > Operations > Diagnosis > Index Manager. Goto Tools > Operations > Diagnosis > Query Performance > Slow Queries. Select a Query and Click on Explain.
